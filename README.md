# Andafar Landing Page

Landing page for **Andafar** - a GIS field data collection platform for non-technical agents.

## Stack

- Plain HTML / CSS / JavaScript (no build step)
- GIS canvas animation (nodes, polylines, cursor repulsion)
- FR / EN language toggle
- Hosted on [Netlify](https://www.netlify.com/)

## Structure

```
index.html          Main landing page
styles.css          All styles
main.js             GIS canvas + site logic + FR/EN toggle
For_landingpage/    Image assets used in the page
logo.png            Andafar logo
```

## Local development

Open `index.html` directly in a browser - no build step required.

## Deploy

Connected to Netlify via GitHub. Every push to `main` triggers an automatic deploy.
